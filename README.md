# 🧩 Pattern Programs (Java)

This repository contains a collection of **pattern programs implemented in Java**.  
It is designed to help beginners improve **logic building, loops, and problem-solving skills**.

---


## 📁 Project Structure
```
├──number_patterns
    ├── simple.java
    ├── reverse_num.java
    ├── Floyd’s Triangle
├── star_patterns/
    ├── Square.java
    ├── Right_angled_triangle.java
    ├── inverted_right_angled.java
```
